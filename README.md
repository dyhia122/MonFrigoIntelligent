Objectif
Permettre une gestion simple, rapide et intelligente des aliments pour réduire le gaspillage et mieux organiser son frigo.

Public visé
-Étudiants
-Familles
-Personnes souhaitant mieux gérer leur alimentation

Fonctionnalités
-Ajouter / modifier / supprimer des aliments
-Suivi des dates de péremption
-Barre de progression (OK / bientôt périmé / périmé)
-Scanner de produits
-Notifications d’expiration
-Statistiques
-Corbeille
-Compte utilisateur moderne

Technologies
-Kotlin, Jetpack Compose
-Room Database
-CameraX, ML Kit (scan)
-Notifications (AlarmManager / WorkManager)
-Material Design 3
